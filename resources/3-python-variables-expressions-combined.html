Aliases üîó

Two Names, Same Object

fruits
‚Üí
["apple", "banana"]
‚Üê
my_list
fruits = ["apple", "banana"]
my_list = fruits  # Alias!

my_list.append("orange")

print(fruits)   # ['apple', 'banana', 'orange']
print(my_list)  # ['apple', 'banana', 'orange']
‚ö†Ô∏è Warning: Aliases share changes!

Copy vs Alias ‚ö†Ô∏è

‚ùå Alias (Shared)

original = [1, 2, 3]
copy = original

copy.append(4)

print(original)
# [1, 2, 3, 4]
# Changed!

‚úì Real Copy (Independent)

original = [1, 2, 3]
copy = original.copy()

copy.append(4)

print(original)
# [1, 2, 3]
# Unchanged!

Part 2: Expressions & Statements

Understanding How Python Code Works

What is an Expression? üí≠

An expression is code that produces a VALUE 
# All of these are expressions
5 + 3                    # ‚Üí 8
10 * 2                    # ‚Üí 20
"Hello " + "World"      # ‚Üí "Hello World"
len("Python")           # ‚Üí 6
7 > 5                    # ‚Üí True
[1, 2, 3]                # ‚Üí [1, 2, 3]

Key Point

Expressions can be replaced by their resulting value

Examples

x = 5 + 3 is same as x = 8

Types of Expressions üéØ

Arithmetic

10 + 5
20 - 3
4 * 6
15 / 3

Comparison

5 > 3
10 == 10
7 != 5
8 >= 8

Logical

True and False
True or False
not True
# String operations
name = "Alice"
greeting = "Hello " + name    # "Hello Alice"

# Function calls
length = len("Python")       # 6
upper = name.upper()          # "ALICE"

What is a Statement? üìù

A statement is an INSTRUCTION that Python executes 
# Assignment Statement
x = 10

# Print Statement
print("Hello")

# If Statement
if x > 5:
    print("Big number")

# Loop Statement
for i in range(3):
    print(i)

# Import Statement
import math
Key Difference: Statements DO something, expressions ARE something (a value)

Expression vs Statement ü§î

Expression

Produces a VALUE

# All produce values
5 + 3
len("hi")
10 > 5
"Hello"

Statement

Performs an ACTION

# All do something
x = 10
print("Hi")
if x > 5:
    pass
Important:
  • ALL expressions are statements
  • NOT all statements are expressions
  • Assignment (x = 5) is a statement, NOT an expression

Simple Example üé®

# Calculate area of a circle

# This is a STATEMENT (assignment)
radius = 5

# This is an EXPRESSION (3.14159 * 5 * 5)
# The whole line is a STATEMENT (assignment)
area = 3.14159 * radius * radius

# This is a STATEMENT (print)
# f"Area is {area}" is an EXPRESSION
print(f"Area is {area}")

Breaking It Down

Expressions in the code:
  • 5
  • 3.14159 * radius * radius
  • f"Area is {area}"
Statements in the code:
  • radius = 5
  • area = ...
  • print(...)

Assignment is NOT an Expression ‚ö†Ô∏è

‚ùå These DON'T Work in Python!

# ‚ùå Can't use assignment as expression
if (x = 5):           # ERROR!
    print("Five")

# ‚ùå Can't assign to assignment
(x = 3) = 5         # ERROR!

‚úì Correct Way

# ‚úì Assign first, then check
x = 5
if x == 5:          # Use == for comparison!
    print("Five")
Remember: Use = for assignment, == for comparison!

Compound Assignment Operators üîß

Shorthand for common operations

Long Way

score = 10
score = score + 5

count = 20
count = count - 3

total = 5
total = total * 2

Short Way ‚ú®

score = 10
score += 5        # Same as score = score + 5

count = 20
count -= 3        # Same as count = count - 3

total = 5
total *= 2        # Same as total = total * 2
# All compound operators
x += 5   # Add
x -= 3   # Subtract
x *= 2   # Multiply
x /= 4   # Divide
x %= 3   # Modulus

Conditional Expression (Ternary) üîÄ

Choose a value based on a condition

Old Way (4 lines)

age = 17

if age >= 18:
    status = "adult"
else:
    status = "minor"

New Way (1 line) ‚ú®

age = 17

status = "adult" if age >= 18 else "minor"

Syntax:

variable = value_if_true if condition else value_if_false
# More examples
grade = "Pass" if score >= 60 else "Fail"

max_num = a if a > b else b

message = "Even" if num % 2 == 0 else "Odd"

Operator Precedence üìä

Order matters! (Like PEMDAS in math) 
# Without precedence, this could be different!
result = 10 + 3 * 2
print(result)  # 16 (not 26!)
# Because * comes before +

# Use parentheses to be clear!
result1 = (10 + 3) * 2    # 26
result2 = 10 + (3 * 2)    # 16

Precedence Order (High to Low)

1. Parentheses ()

2. Exponent **

3. Multiplication/Division * / // %

4. Addition/Subtraction + -

5. Comparison < > <= >= == !=

6. Logical NOT not

7. Logical AND and

8. Logical OR or

Real World Example üåü

# Calculate student's final grade

# Get input (statements)
quiz1 = float(input("Quiz 1 score: "))
quiz2 = float(input("Quiz 2 score: "))
exam = float(input("Exam score: "))

# Calculate average (expression in a statement)
average = (quiz1 + quiz2 + exam) / 3

# Determine letter grade (conditional expression)
letter = "A" if average >= 90 else \
         "B" if average >= 80 else \
         "C" if average >= 70 else \
         "D" if average >= 60 else "F"

# Display result (statement)
print(f"Average: {average:.1f}")
print(f"Grade: {letter}")

Common Mistakes to Avoid üö´

‚ùå Mistake #1

# Using = instead of ==
if x = 5:  # ‚ùå Wrong!
    print("Five")

# Should be:
if x == 5:  # ‚úì Correct!
    print("Five")

‚ùå Mistake #2

# Forgetting precedence
x = 10 + 5 * 2  # 20, not 30!

# Use parentheses:
x = (10 + 5) * 2  # 30 ‚úì

‚ùå Mistake #3

# Division by zero
result = 10 / 0  # ‚ùå Error!

# Check first:
if divisor != 0:
    result = 10 / divisor

‚ùå Mistake #4

# Integer vs Float division
print(7 / 2)   # 3.5 (float)
print(7 // 2)  # 3 (integer)

# Know the difference!
