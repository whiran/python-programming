Say "Hello World!"

  • It is a tradition (since the C programming language book) to display "Hello World!" in a computer console (terminal).
  • print is a Python built-in function name. A function performs some tasks.
  • Here it prints the message (a string) "Hello World!" to the console.
print("Hello World!")
print("Hello World!") Hello World!

Function Call

  • A function, like a computer, processes its input and performs some tasks.
  • A function name followed by a pair of parentheses is a function call ‚Äì the computer runs/executes the function. For example, print("Hello World").
  • The string "Hello World!" is the input to the print function.
  • A string is written inside a pair of double quotes or single quotes: print('Hello World!').
Function = Input ‚Üí Processing ‚Üí Output

Two Execution Modes

You run Python code in one of two modes:

1. Interactive

Type and run code directly inside the Python Interpreter (>>>). Convenient for small snippets.

>> print("Hi!")

2. Script

Type code into a file (.py), then run it as python3 filename.py. Used for most applications.

# script.py
print("Hi!")

More Computations

Python supports common mathematic operations. Use # for comments (ignored by Python).

1 + 2      # Addition
5 - 3      # Subtraction
3 * 5 - 2  # Multiplication
7 / 5      # Division
2 ** 10    # Exponential (Power)
7 // 5     # Quotient (Floor Division)
7 % 5      # Remainder (Modulus)

Operation Orders

It is a best practice to use parenthesis for operation order to avoid ambiguity.

(3 * 5) - 2      # Clear order
3 * (5 - 2)      # Change order
2 ** 3 ** 2      # Unclear
(2 ** 3) ** 2    # No ambiguity
(3 * 5) - 2 13
3 * (5 - 2) 9

Comparison

Comparison results are Boolean values: True or False.

3 > 2           # True
3 == 2          # False
3 != 2          # True
(7 / 5) >= 1    # True
# Can be chained:
2 > math.sqrt(3) > math.sqrt(2)

Logical Operations

Three basic operations: not, and, and or.

not (3 == 2)                # True
(3 > 2) and ((7 / 5) > 3)   # False
(3 == 2) or (3 >= 2)        # True

Variables

A Python variable is a symbolic name that is a reference to data.

pi = 3.14159
diameter = 2 * radius
Variable = Label pointing to data in memory

A Simple Program

pi = 3.14159
radius_input = input("Please input radius: ")
radius = float(radius_input)
diameter = 2 * radius
circumference = pi * diameter
print(circumference)

Better Output (f-strings)

F-strings create text from variables using {variable}.

print(f"The circumference of radius {radius} is {circumference}.")

Format modifiers: :.2f shows 2 decimal places.

print(f"Value: {circumference:.2f}")

Life is Not Linear (Branches)

Python uses indentation (4 spaces) to mark a code block.

if score >= 900:
    grade = "A"
elif score >= 800:
    grade = "B"
else:
    grade = "C"

Life is Full of Repetitions (Loops)

while score < 900:
    print("Please keep studying...")
    score = int(input("New score: "))
While Loop: Repeats code as long as condition is True

List

A sequence of values in square brackets. Access items by index (starting at 0).

scores = [930, 790, 367]
print(scores[0])  # 930
print(scores[1])  # 790

For Loop

Used to iterate over a list.

for score in scores:
    if score >= 900:
        print("Great!")
For Loop: Processes each item in a sequence one by one

Dictionary

Items stored as key: value pairs.

student = {"Name": "Alice", "Age": 20}
print(student["Name"])  # Alice
print(student["Age"])   # 20

Define a Function

Used to avoid redundant code and make logic reusable.

def get_circumference(radius):
    return 2 * math.pi * radius

# Using the function
result = get_circumference(5)
print(result)
